C++基础测试（闭卷）

命题人：Vinkle

本卷为闭卷测试，做本卷时请自觉不要使用编译器，不要到网上搜索答案，也不要翻阅参考书。

本卷所有编译运行结果均来源于 gcc 8.1.0 ，编译参数为g++ -std=c++11 -O0 -Wall。

选择题

选出符合题意的选项，正确答案可能不止一个

1. 以下初始化 int 类型变量正确的是（ ）

A. int n = 0;

B. int n = {0};

C. int n(0);

D. int n{0};

2. 下面关于引用的叙述正确的是（ ）

A. 引用必须被初始化

B. 常引用 const _Type& (_Type 指对象类型) 必须用常量对象初始化

C. 常量对象能直接初始化一般引用

D. 更改非常引用的值，引用所绑定的对象的值也会改变

3.下列对于指针定义及初始化正确的是（ ）

A. int* p = nullptr;

B. int const* p = 123;

C. int* const p; p = NULL;

D.

int n = 1;
int* p1 = &n;
int* const p2 = p1;
int const* p3 = p2;
const int* const p4 = p3;
4. 阅读以下程序，选择正确的选项（ ）
#include <iostream>
using namespace std;
int getLen(int* a, int* b)
{
    return b - a;
}
int main()
{
	int arr[100];
    cout << getLen(begin(arr),end(arr)) << endl;
    return 0;
}

A. getLen 返回的值大小与 sizeof(arr) 相等

B. getLen 返回的值大小与 100*sizeof(int) 相等

C. getLen 返回的值大小与 100*sizeof(int*) 相等

D. getLen 返回的值为 100

5. 想要程序中的 cin 调用 std::cin 我们可以在调用前写（ ）语句

A. using namespace std;

B. using namespace std::cin;

C. using std::cin;

D. using cin = std::cin;

6. 字符串 str 的定义为 std::string str(10,'a')，则下面说法正确的是 ( )

A. str.size() 的值为 $10$

B. str == "a" 这条语句为真

C. str[1] == '\0' 这条语句为真

D. 此定义不合法
